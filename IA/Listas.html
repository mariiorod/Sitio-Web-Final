¿Qué es una lista y cómo usarlas en Prolog?

En Prolog, una lista es una estructura de datos que permite agrupar varios elementos dentro de un solo objeto. Las listas se escriben entre corchetes [] y sus elementos están separados por comas.

Por ejemplo:

[1, 2, 3, 4]

[a, b, c]

Las listas se usan comúnmente en consultas, reglas y manipulaciones, y permiten recorrer o modificar múltiples datos de manera lógica.

 

Descomposición de listas: Cabeza y Cola

Una lista puede dividirse en dos partes:

·       Cabeza (Head): Es el primer elemento de la lista.

·       Cola (Tail): Es el resto de la lista (todos los elementos después del primero).

Se representa con la notación:

 [X | Xs] = [1, 2, 3, 4].

Resultado:

X = 1

Xs = [2, 3, 4]

·       Si se usa _ en lugar de X, se ignora la cabeza.

·       Si se usa _ en lugar de Xs, se ignora la cola.

 

Uso de select para manipular listas

El predicado select se utiliza para buscar un elemento en una lista, remplazarlo por otro y obtener la nueva lista como resultado.

Sintaxis básica:

select(ElementoBuscado, ListaOriginal, ElementoNuevo, Resultado).

Ejemplo:

select(a, [a, b, c], x, Nuevalista).

Resultado:

·       Aquí, a fue encontrado y reemplazado por x.

·       Si no se indica el elemento de reemplazo, el elemento simplemente desaparece.

select(a, [a, b, c], Resultado).

Búsquedas múltiples anidadas

Para realizar más de una sustitución en una misma línea, se pueden anidar las llamadas a select, como:

select(a, [a, b, c], x, R1), select(b, R1, y, ResultadoFinal).

Esto reemplaza a por x, luego b por y.

Resultado:

Con las listas, Prolog puede:

·       Acceder a elementos fácilmente usando descomposición [X | Xs].

·       Reemplazar elementos con select.

·       Eliminar elementos si no se especifica reemplazo.

·       Hacer varias modificaciones anidadas para trabajar con listas más complejas.
