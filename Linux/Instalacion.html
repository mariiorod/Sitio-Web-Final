Instalación de Linux (Ubuntu)

Antes de comenzar el proceso de instalación, es importante contar con una memoria USB booteable que contenga la imagen del sistema operativo (ISO) y haber configurado previamente la secuencia de arranque en la BIOS para que el equipo inicie desde la USB.

 

1. Seleccionar Instalar

Al encender el equipo con la USB conectada, se mostrará un menú inicial conocido como GRUB.

El GRUB (GRand Unified Bootloader) es un cargador de arranque que permite elegir cómo iniciar el sistema. Aquí se podrá seleccionar entre las opciones de:

·       Instalar/ Probar Ubuntu

·       Instalar en modo de gráficos seguros

·       Modo OEM, entre otros.

¿Qué es el modo OEM?

El modo OEM (Original Equipment Manufacturer) es una opción pensada principalmente para fabricantes o técnicos que desean preinstalar el sistema operativo en una computadora antes de entregársela al usuario final. Permite hacer la instalación, personalizar el sistema (por ejemplo, añadir drivers, programas o configuraciones) y luego dejarlo listo para que, cuando el usuario encienda el equipo por primera vez, complete su configuración personal (nombre de usuario, contraseña, etc.).

Seleccionamos Instalar Ubuntu para comenzar.

 

2. Elegir idioma y seleccionar Instalar

Se mostrará una pantalla para seleccionar el idioma del sistema. Tras elegirlo, damos clic en Instalar Ubuntu.

 

3. Disposición del teclado

Seleccionamos la distribución del teclado que corresponda a nuestro idioma o preferencia.

 

4. Tipo de instalación

Se podrá elegir entre:

·       Instalación normal: incluye software de oficina, navegador y utilidades.

·       Instalación mínima: solo el navegador web y herramientas básicas.

Adicionalmente, se puede optar por:

·       Descargar actualizaciones mientras se instala.

·       Instalar software de terceros para hardware adicional (recomendado si se usa Wi-Fi o tarjetas gráficas propietarias).

 

5. Seleccionar Más opciones

Para tener mayor control sobre el lugar y forma en que se instalará Linux, se recomienda elegir Más opciones.

 

6. Gestión de discos y particiones

Se mostrará la lista de discos disponibles.

Si el disco está vacío, debe seleccionarse y hacer clic en Nueva tabla de particiones. Aparecerá un cuadro de diálogo para confirmar esta acción.

 

7. Crear partición raíz

Después de crear la tabla, damos doble clic en el espacio libre para crear una nueva partición:

·       Tamaño: Definimos el espacio a usar.

·       Tipo de partición: Primaria.

·       Ubicación: Al principio del espacio.

·       Sistema de archivos: ext4 transaccional.

¿Qué es ext4 transaccional?

Es un sistema de archivos moderno, rápido y confiable, que protege los datos gracias a su capacidad de manejar transacciones, reduciendo riesgos de corrupción en caso de apagones o errores.

·       Punto de montaje: / (es decir, la raíz del sistema).

 

8. Seleccionar partición e instalar

Se selecciona la partición recién creada y se presiona Instalar ahora.

 

9. Advertencia por falta de partición EFI (si aplica)

Si el sistema detecta que no existe una partición EFI, mostrará una advertencia.

Esto ocurre en sistemas modernos con partición GPT, donde EFI es fundamental para el arranque.

¿Qué es una partición EFI?

Es una partición especial que utilizan las BIOS modernas (UEFI) para arrancar el sistema operativo.

¿Qué pasa si no se crea?

Puede haber errores de arranque después de la instalación, especialmente en discos con formato GPT.

Si el disco está en formato MBR (antiguo), este aviso puede ignorarse con menor riesgo.

 

10. Confirmación de cambios

Si todo está correcto, se mostrará un resumen de las operaciones. Damos clic en Continuar.

 

11. Selección de zona horaria

Seleccionamos la ciudad o región donde reside el usuario para definir la hora local del sistema.

 

12. Creación de usuario

El sistema pedirá:

·       Nombre del usuario

·       Nombre del equipo

·       Contraseña

Además, podrá elegirse entre:

·       Iniciar sesión automáticamente

·       Solicitar contraseña al iniciar sesión (recomendado en equipos compartidos o si se desea mayor seguridad)

También se ofrece la opción de Usar Active Directory, un sistema de autenticación de usuarios común en redes empresariales o educativas.

 

13. Proceso de instalación

Comenzará la instalación del sistema operativo. Solo resta esperar a que termine.

 

14. Reinicio y retiro de la USB

Una vez finalizada la instalación, el sistema solicitará reiniciar.

En ese momento, debe retirarse la memoria USB para que el equipo arranque directamente desde el disco duro.

 

Con esto, la distribución de Linux (Ubuntu) quedará completamente instalada y lista para ser utilizada.
