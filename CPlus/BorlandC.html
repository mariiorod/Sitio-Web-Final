Instalación de Borland C++

¿Qué es Borland C++?

Borland C++ es un entorno de desarrollo integrado (IDE) clásico para el lenguaje C y C++, muy utilizado en décadas pasadas especialmente en entornos educativos. Aunque hoy en día ha sido reemplazado por opciones más modernas, aún se usa por su sencillez y compatibilidad con sistemas antiguos o prácticas académicas básicas.

¿Por qué instalarlo hoy en día?

Borland C++ sigue siendo útil para introducirse en la programación estructurada, especialmente en cursos donde se enseñan fundamentos sin depender de entornos modernos. Además, su entorno minimalista permite centrarse en el código.

 

Instalación de Borland C++

Nota: Al igual que con otros programas, se recomienda ejecutar el instalador como Administrador para evitar errores de permisos.

1.  Ejecutar el instalador

Hacer clic derecho sobre el archivo Setup.exe y seleccionar Ejecutar como administrador.

 

2.  Ventana de introducción

Al comenzar, aparecerá una ventana que ofrece visualizar un archivo .TXT con información sobre la instalación. Se recomienda seleccionar "Skip" para omitirlo y continuar.

 

3.  Elegir el tipo de instalación

Se podrá escoger entre:

a.  Typical: instalación estándar.

b.  Custom: instalación personalizada, recomendada para usuarios con experiencia que deseen ajustar detalles del entorno.

 

4.  Seleccionar la carpeta de destino para Borland

Se elige el directorio donde se instalará el programa principal. Puede dejarse la opción por defecto o elegir otra carpeta.

 

5.  Seleccionar la carpeta para los Borland Database Engine (Componentes del motor de Borland)

Aquí se indica la ubicación donde se instalarán los componentes del motor Engine de Borland. Los Engines son componentes esenciales que permiten la ejecución de programas dentro del entorno de desarrollo integrado (IDE). Estos incluyen bibliotecas, herramientas y archivos necesarios para compilar y ejecutar las aplicaciones). Luego, hacer clic en Next.

 

6.  Seleccionar la ruta para los BDE SDK (Borland Database Engine y Kit de Desarrollo)

Esta ruta será utilizada para los archivos del Borland Database Engine (BDE) y su Kit de Desarrollo (SDK). El BDE permite que Borland interactúe con bases de datos, mientras que el SDK proporciona las herramientas necesarias para integrar soporte de bases de datos en las aplicaciones desarrolladas.. Presionar Next para continuar.

 

7.  Comenzar la instalación

Hacer clic en Install para que se copien los archivos e inicie el proceso.

 

8.  Esperar a que termine la instalación

Durante el proceso, puede aparecer un cuadro de diálogo adicional. En ese caso, simplemente seleccionar Skip para continuar.

 

9.  Finalización

Al concluir, se mostrará la ventana de finalización del proceso.

 

10.              Opción de registro

El sistema preguntará si se desea registrar el producto. Esta acción es opcional. Se puede hacer clic en Skip para omitirla.

 

11.              Listo para usarse

Borland C++ está ahora instalado y disponible para comenzar a programar.
