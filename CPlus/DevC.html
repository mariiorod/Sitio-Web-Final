Instalación de Dev-C++

¿Qué es Dev-C++?

Dev-C++ es un entorno de desarrollo integrado (IDE) para programación en lenguaje C y C++. Es una herramienta ligera y gratuita que permite escribir, compilar y depurar programas, ideal para principiantes en programación estructurada. Dev-C++ incluye un compilador (generalmente MinGW) y una interfaz gráfica simple para facilitar el trabajo con proyectos de código.

¿Para qué sirve instalarlo?

Este IDE se utiliza para escribir y ejecutar programas en C y C++, por lo que es fundamental contar con él correctamente instalado para comenzar a practicar los fundamentos de la programación.

 

Instalación de Dev-C++

Nota: Se recomienda ejecutar el archivo de instalación como Administrador para evitar errores durante la instalación.

1.  Ejecutar el instalador como administrador.

Esto se hace dando clic derecho sobre el archivo descargado y seleccionando "Ejecutar como administrador".

 

2.  Seleccionar el lenguaje de instalación.

Al iniciar, se pedirá elegir el idioma. Se selecciona el español (u otro de preferencia) y se continúa.

 

3.  Aceptar el acuerdo de licencia.

Es necesario aceptar los términos para continuar con el proceso de instalación.

 

4.  Elegir el tipo de instalación.

Se recomienda seleccionar instalación completa (Full) para contar con todas las herramientas del entorno.

 

5.  Seleccionar la carpeta de destino.

Aquí se elige dónde se instalará el programa (por defecto es en Archivos de Programa) y se presiona Instalar.

 

6.  Permitir la instalación para todos los usuarios.

Durante el proceso, se preguntará si se desea instalar para todos los usuarios del sistema. Se recomienda decir Sí.

 

7.  Finalizar la instalación.

Una vez que se copian los archivos, se presiona Terminar.

 

8.  Configurar el programa al primer arranque.

·       Al abrir por primera vez Dev-C++, se pedirá de nuevo seleccionar el idioma. Se elige el mismo que al principio y se presiona Next.

·       Luego se preguntará si se desea activar el autocompletado. Se recomienda Sí, ya que facilita el desarrollo sugiriendo funciones y completando código.

·       Finalmente se preguntará si se desea crear un caché para el autocompletado. También se recomienda Sí.

 

9.  Instalación finalizada.

Después de unos segundos, el entorno estará listo para usarse y podrá comenzar a programar en C o C++.
