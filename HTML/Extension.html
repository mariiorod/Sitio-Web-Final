Extensión de una Página Web y Programa para Crear, Editar y Modificar

Las páginas web suelen tener una extensión que indica el tipo de archivo y su formato. El HTML (HyperText Markup Language) es el lenguaje utilizado para estructurar páginas web, y el archivo que contiene este código tiene la extensión .html. Esto significa que el archivo es un documento de texto que contiene el código HTML necesario para construir la página web.

  • Extensión de archivo: Las páginas web suelen guardarse con la extensión .html (o .htm en algunos casos), lo que indica que el archivo es una página HTML.
  • Programas para crear, editar y modificar páginas web: Existen varias herramientas y programas que permiten crear y modificar páginas web, pero de igual forma, se puede trabajar con Bloc de Notas, un editor de texto simple que viene preinstalado en los sistemas operativos Windows.

¿Por qué usar Bloc de Notas?

Aunque existen programas más avanzados y específicos para el desarrollo web, como Visual Studio Code, Sublime Text o Notepad++, Bloc de Notas es una excelente herramienta para comenzar, ya que es accesible, simple y permite trabajar con archivos HTML sin ninguna complejidad adicional. Su facilidad para guardar archivos con la extensión .html lo convierte en una opción válida para quienes están empezando a crear páginas web.

 

Ejemplo de archivo HTML guardado en Bloc de Notas

1.  Abrir Bloc de Notas.

2.  Escribir el código HTML que se desee.

3.  Para guardar el archivo, seleccionar Archivo  à Guardar como....

4.  En el cuadro de diálogo, escribir el nombre del archivo con la extensión .html (por ejemplo, mi_pagina.html).

5.  Asegurarse de que el tipo de archivo esté configurado como "Todos los archivos".

6.  Hacer clic en Guardar.

Ahora se puede abrir el archivo guardado en cualquier navegador web y ver la página web en acción.
