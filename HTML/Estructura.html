Estructura básica de una página Web

El HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar y presentar contenido en la web. Una página web básica en HTML está compuesta por varias secciones que permiten organizar el contenido de manera eficiente.

La estructura básica de una página HTML incluye los siguientes componentes esenciales:

·       <html>: Es el contenedor principal de la página web. Todo el contenido de la página debe estar dentro de esta etiqueta.

·       <head>: Contiene información sobre el documento, como el título de la página, enlaces a hojas de estilo (CSS), y otros metadatos que no se ven directamente en el contenido visible.

·       <title>: Especifica el título de la página, que aparece en la pestaña del navegador. Este título es importante para la accesibilidad y el SEO (optimización en motores de búsqueda).

·       <body>: Es donde se coloca el contenido visible de la página. Todo el texto, imágenes, enlaces y otros elementos que verán los usuarios deben ir dentro de esta sección.

 

Ejemplo de la Estructura Básica de una Página Web en HTML

<html>

<head>

    <title>Mi Página Web</title>

</head>

<body>

    <h1>Bienvenidos a Mi Página Web</h1>

    <p>Este es el contenido de mi página web.</p>

</body>

</html>

 

Resultado en la página web:

Cuando se carga este código en un navegador, se verá lo siguiente:

·       En la pestaña del navegador aparecerá el texto "Mi Página Web" (lo que definimos en la etiqueta <title>).

·       En el cuerpo de la página, se mostrará un encabezado grande (<h1>) con el texto "Bienvenidos a Mi Página Web".

·       Justo debajo del encabezado, aparecerá un párrafo (<p>) con el texto "Este es el contenido de mi página web.".
